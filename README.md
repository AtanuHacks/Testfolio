# Portfolio Website

A modern, full-stack portfolio website showcasing my skills, projects, and experience with a fully functional backend.

## 🚀 Features

- **Hero Section**: Eye-catching introduction with personal branding
- **About Section**: Detailed information about background and interests
- **Education Section**: Academic qualifications and achievements
- **Skills Section**: Interactive icon grid displaying technical skills
- **Projects Section**: Comprehensive showcase of completed projects
- **Contact Form**: Fully functional form with MongoDB backend integration
- **Messages Dashboard**: Admin page to view all contact form submissions
- **Smooth Animations**: GSAP-powered animations for enhanced user experience
- **Responsive Design**: Optimized for all device sizes

## 🛠️ Tech Stack

### Frontend
- HTML5
- CSS3 (Custom styling, no frameworks)
- JavaScript (ES6+)
- GSAP (GreenSock Animation Platform)

### Backend
- Node.js
- Express.js
- MongoDB (with Mongoose)
- CORS enabled for cross-origin requests
